Yazi 的快捷键逻辑深受 Vim 启发,上手后效率很高。我把最常用的快捷键整理成了分类速查表,方便你随时查阅。

### 🚀 Yazi 快捷键速查表

#### 1. 核心导航

Yazi 的目录导航完全遵循 Vim 的键位逻辑。

| 快捷键       | 功能说明                    | 备注                            |
| :----------- | :-------------------------- | :------------------------------ |
| `h` / `l`    | 返回上级 / **进入**目录     | `l` 也可用 `<Right>` 或 `Enter` |
| `j` / `k`    | 向下 / 向上移动光标         | 也可用方向键                    |
| `gg` / `G`   | 跳到列表**顶部** / **底部** | 快速定位                        |
| `数字` + `G` | 跳转到指定行                | 例如 `5G` 跳转到第5行           |
| `Tab`        | 显示当前文件详细信息        | 查看文件大小、权限等            |

#### 2. 文件操作 (核心操作)

最常用的复制、粘贴、删除操作,逻辑非常清晰。

| 快捷键 | 功能说明                  | 备注                        |
| :----- | :------------------------ | :-------------------------- |
| `y`    | 复制 (Yank) 选中文件      |                             |
| `x`    | 剪切 (Yank--cut) 选中文件 |                             |
| `p`    | 粘贴 (Paste) 文件         |                             |
| `d`    | 删除文件(移至回收站)      |                             |
| `D`    | **永久删除**文件          | 危险操作,跳过回收站         |
| `r`    | 重命名文件                |                             |
| `a`    | 新建文件/文件夹           | 输入名称以`/`结尾则为文件夹 |

#### 3. 选择与模式

当你需要批量操作多个文件时,这些快捷键非常实用。

| 快捷键     | 功能说明               | 备注                |
| :--------- | :--------------------- | :------------------ |
| `Space`    | 切换当前文件的选中状态 | 常用于多选文件      |
| `v`        | 进入视觉选择模式       | 类似 Vim,可连续选中 |
| `Ctrl + a` | 全选所有文件           |                     |
| `Ctrl + r` | 反选所有文件           |                     |

#### 4. 查找与跳转

这几个搜索快捷键是 Yazi 的精髓,能极大提升定位文件的效率。

| 快捷键        | 功能说明                       | 所需工具             |
| :------------ | :----------------------------- | :------------------- |
| `/`           | 在当前目录过滤/查找文件        | 内置                 |
| `s`           | 通过文件名**递归搜索**文件     | 需要 `fd`            |
| `S`           | 通过内容**全文搜索**文件       | 需要 `ripgrep` (rg)  |
| `z`           | 使用 `zoxide` 快速跳转历史目录 | 需要 `zoxide`        |
| `Z`           | 使用 `fzf` 模糊查找文件/目录   | 需要 `fzf`           |
| `~` 或 `<F1>` | 打开帮助菜单                   | 记不住快捷键时可以按 |

#### 5. 标签页与视图

| 快捷键           | 功能说明                   | 备注                    |
| :--------------- | :------------------------- | :---------------------- |
| `t`              | 新建标签页                 |                         |
| `[` / `]`        | 切换到上一个/下一个标签页  |                         |
| `数字` (`0`-`9`) | 直接跳转到指定序号的标签页 |                         |
| `.`              | 显示/隐藏隐藏文件          | 即文件名以`.`开头的文件 |
| `,`              | 切换文件排序方式           | 按时间、大小、名称等    |

### 💡 进阶使用技巧

1. **随时求助: `~` 键**
    Yazi 的帮助系统很强大。当你忘记某个快捷键时,直接按 `~` 键(波浪线)就能打开帮助界面,而且支持 `f` 键进行关键词搜索,不用退出程序去查资料。

2. **复制文件路径: `c` + `c`**
    在需要把文件路径分享给其他人或另一个终端时,先按 `c`,再快速按一次 `c` (连按两下 `c`),即可将当前选中文件的**完整绝对路径**复制到系统剪贴板。

3. **文件预览滚动: `K` / `J`**
    在右侧预览代码或长文本时,使用大写的 `K` 和 `J` 可以**向上/向下滚动预览窗口**,而不用移动光标去选中文件。

4. **退出方式的选择: `q` vs `Q`**
    - 直接按 `q` 退出 Yazi,你会停留在启动时的目录。
    - 如果你希望在退出后,**终端的当前目录**自动切换到 Yazi 最后浏览的目录,需要配置 Shell 集成,然后按 `Q` 退出。
